API Key Instructions

Create an API key at https://account.arena.net/applications and paste it in the "API Key" input box above in order to view your GW2 account's data.
You may have multiple accounts or API keys, so give them a name to help you remember.

Privacy Statement

Last changed on: 2016-02-01

Knowing an API key allows anyone to view your account's ingame information, such as: characters and guilds, bank and inventory contents, gold and assets, achievements and unlocks, and Trading Post and PvP match history. You may limit which of these type of information is exposed when creating an API key, and remove access to these information by deleting the API key at the ArenaNet website.

GW2Timer.com does not view, store, edit, collect, or datamine API keys or the information API keys provide. When you enter the API key into the website, you are performing calculations locally on your computer, and the keys are stored in your web browser, not a server.

GW2Timer.com uses Google Analytics and is able to view and store text that is shown on your address bar when you are on this website. If you do not wish to expose your API key, do not put your API key in the browser address bar. To share your key with other people, private message them the key and direct them to gw2timer.com/account
